# صفحة الخدمات الأخرى + حاسبة البلاط

## الهدف
إنشاء صفحة "خدمات أخرى" تجمع الأدوات الفرعية تحت مكان واحد، 
وإضافة حاسبة بلاط وسيراميك بوضعين (عادي ومتقدم).

---

## الجزء 1: زر "خدمات أخرى" في Header

### الموقع
في header الكتالوج، بجانب زر "كتالوج تفاعلي" (نفس التنسيق والحجم).

### التفاصيل
- النص: "خدمات أخرى"
- الأيقونة: `Grid3x3` أو `LayoutGrid` من lucide-react
- نفس styling زر "كتالوج تفاعلي" (border ذهبي، خلفية شفافة، نص أبيض)
- عند الضغط: ينتقل إلى `/services`

### الإجراء المطلوب
1. إزالة زر "+ اقتراح منتج" من header (سينتقل لصفحة الخدمات)
2. إضافة زر "خدمات أخرى" بجانب "كتالوج تفاعلي"

## الجزء 3: حاسبة البلاط (TileCalculatorModal)

### الشكل العام

Modal كبير (max-width: 700px) يفتح فوق صفحة الخدمات. بداخله:
- Header: "حاسبة البلاط والسيراميك" + زر إغلاق (X)
- Tabs: "عادية" | "متقدمة"
- محتوى الحاسبة حسب الـ tab النشط

### مقاسات البلاط الجاهزة

### الحاسبة العادية (BasicCalculator.tsx)

#### المدخلات:
1. **مقاس البلاطة** (TileSizeSelector — grid من buttons)
   - إذا اختار "مخصص": يظهر input للطول والعرض بالسنتيمتر
2. **الطول** (input رقمي، بالمتر)
3. **العرض** (input رقمي، بالمتر)

#### الثوابت:
```typescript
const DEFAULT_WASTE_PERCENT = 10; // ثابت، لا يظهر للمستخدم
```

#### المعادلات:
```typescript
const tileAreaM2 = (tileWidth * tileHeight) / 10000;
const roomArea = length * width;
const areaWithWaste = roomArea * 1.10;
const tilesNeeded = Math.ceil(areaWithWaste / tileAreaM2);
```

#### النتيجة:
كرت يعرض:
- 📐 مساحة الغرفة: X m²
- 📦 مع الهدر (10%): X m²
- 🎯 تحتاج: X بلاطة

### الحاسبة المتقدمة (AdvancedCalculator.tsx)

#### المدخلات:
1. **مقاس البلاطة** (نفس TileSizeSelector)
2. **نوع التركيب** (radio buttons):
   - أرضية
   - جدار
   - الاثنين
3. **الطول** (متر)
4. **العرض** (متر)
5. **الارتفاع** (متر) — يظهر فقط مع "جدار" أو "الاثنين"
6. **عدد الأبواب** (input رقمي، default: 0)
7. **عدد النوافذ** (input رقمي، default: 0)
8. **نسبة الهدر** (select):
   - 10% — تركيب عادي
   - 15% — تركيب قطري
   - 20% — تصاميم معقدة
   - مخصص → input رقمي

#### الثوابت:
```typescript
const DOOR_AREA = 1.76;   // m² (0.84 × 2.10)
const WINDOW_AREA = 1.5;  // m² (1.0 × 1.5 متوسط)
```

#### المعادلات:
```typescript
const tileAreaM2 = (tileWidth * tileHeight) / 10000;

// مساحة الأرضية
const floorArea = length * width;

// مساحة الجدران
const wallsArea = 2 * (length + width) * height;

// خصم الفتحات
const openingsArea = (doors * DOOR_AREA) + (windows * WINDOW_AREA);
const netWallsArea = Math.max(0, wallsArea - openingsArea);

// الحسابات حسب نوع التركيب
let floorTiles = 0;
let wallTiles = 0;
let floorWithWaste = 0;
let wallWithWaste = 0;

if (type === 'floor' || type === 'both') {
  floorWithWaste = floorArea * (1 + wastePercent / 100);
  floorTiles = Math.ceil(floorWithWaste / tileAreaM2);
}

if (type === 'walls' || type === 'both') {
  wallWithWaste = netWallsArea * (1 + wastePercent / 100);
  wallTiles = Math.ceil(wallWithWaste / tileAreaM2);
}

const totalTiles = floorTiles + wallTiles;
const totalArea = floorWithWaste + wallWithWaste;
```

#### النتيجة (عرض مفصل):

قسم "تفاصيل الغرفة":
- الأبعاد
- مساحة الأرضية
- مساحة الجدران
- خصم الفتحات
- المساحة الصافية

قسم "الكميات المطلوبة" (يظهر حسب نوع التركيب):
- الأرضية: مساحة + عدد البلاطات
- الجدران: مساحة + عدد البلاطات

قسم "الإجمالي":
- المساحة الكلية
- إجمالي البلاطات

## Validation

- الأرقام فقط في الـ inputs الرقمية
- لا قيم سالبة
- الحد الأدنى: 0.1 متر
- الحد الأعلى: 100 متر
- المقاس المخصص: بين 5cm و 200cm
- إذا المدخلات غير صحيحة أو فارغة، النتيجة تعرض "—" بدلاً من رقم

---

## التفاعل

- **الحساب فوري** — بدون زر "احسب"
- **Animation ناعم** عند تغيير النتيجة (Framer Motion: opacity + 
  translate-y صغير)
- **Tabs** تحفظ حالتها عند التبديل (يعني لو ملأت الحاسبة العادية 
  وبدّلت للمتقدمة ورجعت، البيانات موجودة)

## التصميم

### الألوان:
- Primary: `#1B3D4F`
- Secondary: `#2A5A75`
- Background Modal: `#FAF7F2`
- Background inputs: white
- Border: `#E8DFD0`
- Text dark: `#1B3D4F`

### الخطوط:
- IBM Plex Sans Arabic أو Tajawal
- الوزن 500 للعناوين، 400 للنصوص

### RTL:
- الصفحة كلها RTL
- الأرقام في الـ inputs LTR (عادي للأرقام)

### Responsive:
- Modal: max-width 700px على الديسكتوب، full-screen على الموبايل
- Grid المدخلات: عمودين على الديسكتوب، عمود واحد على الموبايل
- Tabs: scrollable على الموبايل
