# تطوير موديول توليد الصور — دعم جميع أنواع المنتجات
# Decorative Generator Upgrade — Support All Product Types

---

## 📋 ملخص التعديل

النظام الحالي يدعم فقط **تركيب الخامات على الأسطح** (سيراميك على أرضية).
التعديل المطلوب: دعم **3 أوضاع توليد مختلفة** حسب نوع المنتج.

---

## 🎯 الأوضاع الثلاثة المطلوبة

### 1️⃣ وضع الأسطح (Surface Mode)
**المنتجات:** سيراميك، بورسلان، باركيه، ورق جدران، موكيت، بلاط مسابح، بلاط ديكور، أرضيات بلاستيك خشبي

**طريقة التوليد:** تركيب الخامة على أرضية أو جدار غرفة

**مثال Prompt:**
```
"...featuring this marble tile as the floor covering..."
```

### 2️⃣ وضع المنتجات (Product Mode)
**المنتجات:** خلاطات، شاور، دش، جاكوزي، مغاسل، أدوات صحية، مظلات، جلسات خارجية، عشب صناعي، أنظمة واجهات

**طريقة التوليد:** وضع المنتج في مشهد واقعي (حمام، مطبخ، فناء)

**مثال Prompt:**
```
"...with this chrome faucet installed on the kitchen counter..."
```

### 3️⃣ وضع العرض (Showcase Mode)
**المنتجات:** غراء، ترويبة، سيليكون، بروفايل، زوايا ديكور، فواصل تمدد، مواد البناء المعبأة

**طريقة التوليد:** صورة منتج احترافية على خلفية نظيفة

**مثال Prompt:**
```
"...professional product photography of this adhesive package on clean studio background..."
```

### 1. Backend: `openai_service.py`

**التعديل:** تحديث الـ System Prompt ليُرجع `generation_mode` إضافة للتحليل الحالي

```python
# openai_service.py

ANALYSIS_SYSTEM_PROMPT = """
You are an expert in building materials and interior design products.
Analyze the uploaded product image and return a JSON object with:

1. product_type: نوع المنتج بالعربي
2. product_type_en: Product type in English
3. color: اللون بالعربي
4. color_en: Color in English
5. pattern: النمط بالعربي (إن وجد)
6. pattern_en: Pattern in English (if any)
7. surface: نوع السطح بالعربي
8. surface_en: Surface type in English
9. description_en: Full English description for image generation
10. generation_mode: One of ["surface", "product", "showcase"]
11. recommended_placement: Where this product should appear (floor, wall, counter, outdoor, studio)
12. product_category: Main category for filtering options

GENERATION MODE RULES:
- "surface": For materials that cover surfaces (tiles, ceramics, porcelain, parquet, wallpaper, carpet, pool tiles, vinyl flooring)
- "product": For standalone products placed IN a scene (faucets, showers, bathtubs, jacuzzi, sinks, toilets, outdoor furniture, umbrellas, artificial grass, facade systems)
- "showcase": For packaged/construction materials (adhesive, grout, silicone, profiles, corner trims, expansion joints, waterproofing)

RECOMMENDED_PLACEMENT VALUES:
- For surface mode: "floor", "wall", "both"
- For product mode: "bathroom", "kitchen", "outdoor", "pool_area", "entrance"
- For showcase mode: "studio"

PRODUCT_CATEGORY VALUES:
- "flooring": أرضيات
- "wall_covering": جدران
- "sanitary": أدوات صحية
- "outdoor": خارجي
- "construction": مواد بناء

Return ONLY valid JSON, no markdown or explanations.
"""

def analyze_product_image(image_url: str) -> dict:
    """Analyze product image and determine generation mode"""
    response = openai_client.chat.completions.create(
        model="gpt-4o-mini",
        messages=[
            {"role": "system", "content": ANALYSIS_SYSTEM_PROMPT},
            {
                "role": "user",
                "content": [
                    {"type": "text", "text": "Analyze this building material/product image:"},
                    {"type": "image_url", "image_url": {"url": image_url}}
                ]
            }
        ],
        response_format={"type": "json_object"}
    )
    
    return json.loads(response.choices[0].message.content)
```

# kie_ai_service.py

def build_prompt(analysis: dict, settings: dict) -> str:
    """Route to appropriate prompt builder based on generation mode"""
    mode = analysis.get("generation_mode", "surface")
    
    if mode == "surface":
        return build_surface_prompt(analysis, settings)
    elif mode == "product":
        return build_product_prompt(analysis, settings)
    elif mode == "showcase":
        return build_showcase_prompt(analysis, settings)
    else:
        return build_surface_prompt(analysis, settings)  # fallback


def build_surface_prompt(analysis: dict, settings: dict) -> str:
    """Prompt for tiles, flooring, wallpaper - materials that cover surfaces"""
    
    placement = analysis.get("recommended_placement", "floor")
    placement_text = "floor covering" if placement == "floor" else "wall covering" if placement == "wall" else "surface covering"
    
    return f"""
{QUALITY_PREFIX[settings.get('render_quality', 'standard')]},
interior design photography of {SPACE_TYPES[settings['space_type']]},
featuring {analysis['description_en']} as the main {placement_text},
{DESIGN_STYLES[settings['design_style']]},
{LIGHTING_OPTIONS[settings['lighting']]},
{CAMERA_ANGLES[settings['camera_angle']]},
wide-angle architectural lens,
detailed material texture visible, natural material appearance,
{MOODS[settings['mood']]},
photorealistic, architectural visualization, professional interior photography
""".strip().replace('\n', ' ')


def build_product_prompt(analysis: dict, settings: dict) -> str:
    """Prompt for faucets, showers, furniture - products placed IN a scene"""
    
    placement = analysis.get("recommended_placement", "bathroom")
    
    # Map placement to appropriate scene
    scene_map = {
        "bathroom": "modern luxury bathroom",
        "kitchen": "contemporary kitchen",
        "outdoor": "elegant outdoor patio",
        "pool_area": "luxury pool area",
        "entrance": "grand entrance lobby"
    }
    scene = scene_map.get(placement, "modern interior space")
    
    return f"""
{QUALITY_PREFIX[settings.get('render_quality', 'standard')]},
interior design photography of {scene},
with this {analysis['description_en']} professionally installed and featured prominently,
the product is the focal point of the image,
{DESIGN_STYLES[settings['design_style']]},
{LIGHTING_OPTIONS[settings['lighting']]},
{CAMERA_ANGLES[settings['camera_angle']]},
product clearly visible and well-lit,
{MOODS[settings['mood']]},
photorealistic, architectural visualization, professional interior photography,
the product maintains its exact appearance from the reference image
""".strip().replace('\n', ' ')

product centered and clearly visible,
sharp focus on product details and packaging,
high-end commercial photography style,
clean minimal composition,
no distracting elements, product is the sole focus
""".strip().replace('\n', ' ')

### 3. Backend: `views.py`

**التعديل:** تعديل `generate()` ليمرر الـ `generation_mode` ويعدّل الـ behavior

```python
# views.py - في دالة generate()

@api_view(['POST'])
@permission_classes([IsSuperAdmin])
def generate(request):
    generation_id = request.data.get('generation_id')
    settings = request.data.get('settings', {})
    
    try:
        generation = DecorativeGeneration.objects.get(
            id=generation_id,
            created_by=request.user
        )
        
        if not generation.vision_analysis:
            return Response(
                {"error": "يجب تحليل الصورة أولاً"},
                status=status.HTTP_400_BAD_REQUEST
            )
        
        analysis = generation.vision_analysis
        generation_mode = analysis.get('generation_mode', 'surface')
        
        # Build prompt based on mode
        prompt = build_prompt(analysis, settings)
        
        # For showcase mode, we might not send the image as reference
        # depending on the product type
        if generation_mode == "showcase":
            # For packaged products, send image as reference to maintain appearance
            image_input = [generation.source_image_url]
        else:
            # For surface and product modes, always use image as reference
            image_input = [generation.source_image_url]
        
        # Create Kie.ai task
        task_response = kie_ai_service.create_generation_task(
            prompt=prompt,
            image_urls=image_input,
            aspect_ratio=settings.get('aspect_ratio', '16:9'),
            resolution=RESOLUTION_MAP[settings.get('render_quality', 'standard')]
        )
        
        # Save to database
        generation.generation_settings = {
            **settings,
            'generation_mode': generation_mode
        }
        generation.prompt_used = prompt
        generation.kie_task_id = task_response['data']['taskId']
        generation.status = 'generating'
        generation.save()
        
        return Response({
            "generation_id": generation.id,
            "task_id": generation.kie_task_id,
            "mode": generation_mode,
            "prompt": prompt
        })
        
    except DecorativeGeneration.DoesNotExist:
        return Response(
            {"error": "السجل غير موجود"},
            status=status.HTTP_404_NOT_FOUND
        )
```

## 🧪 اختبار التغييرات

### سيناريوهات الاختبار:

1. **رفع صورة سيراميك** → يجب أن يكتشف `mode: surface` ويعرض خيارات الأرضيات/الجدران

2. **رفع صورة خلاط** → يجب أن يكتشف `mode: product` ويعرض خيارات الحمام/المطبخ

3. **رفع صورة علبة غراء** → يجب أن يكتشف `mode: showcase` ويخفي خيارات نوع الغرفة

4. **اختيار قالب غرفة** → يجب أن يملأ الإعدادات الافتراضية تلقائياً

## ⚠️ ملاحظات مهمة

1. **لا تغيّر** الـ API endpoints أو هيكل الـ Response الحالي
2. **أضف** الحقول الجديدة للـ `vision_analysis` بدون حذف الحقول القديمة
3. **الوضع الافتراضي** دائماً `surface` للتوافق مع السجلات القديمة
4. **الـ Prompts** في `kie_ai_service.py` يجب أن تكون بالإنجليزية فقط
5. **اختبر** مع صور فعلية من كل فئة قبل الـ deploy
