# إضافة كرت "تواصل معنا" + صفحة التواصل

## السياق
صفحة `/services` فيها حالياً 5 خدمات (اقتراح منتج، حاسبة البلاط، 
حاسبة الباركيه، حاسبة الغراء، فروعنا).

المطلوب: إضافة كرت سادس "تواصل معنا" يفتح صفحة جديدة `/contact` 
تعرض كل قنوات التواصل الرسمية لبيت الإباء.

## الجزء 1: الكرت في صفحة الخدمات

### بيانات الكرت

أضف هذا الكرت في نهاية قائمة الخدمات في `ServicesPage.tsx`:
```jsx
{
  icon: Phone,
  title: 'تواصل معنا',
  description: 'تواصل مع بيت الإباء عبر القنوات الرسمية، الدعم الفني، أو شبكاتنا الاجتماعية',
  actionLabel: 'اتصل بنا',
  action: () => navigate('/contact'),
}
```

استخدم نفس تصميم ServiceCard الموجود — لا حاجة لتغييرات.

### تصميم ContactChannelCard.tsx
```jsx
import { motion } from 'framer-motion';
import { Phone, MessageCircle, Mail } from 'lucide-react';

const iconMap = {
  Phone,
  MessageCircle,
  Mail,
};

export default function ContactChannelCard({ channel, delay = 0 }) {
  const Icon = iconMap[channel.icon];
  
  return (
    <motion.a
      href={channel.href}
      target={channel.type === 'email' ? undefined : '_blank'}
      rel="noopener noreferrer"
      initial={{ opacity: 0, y: 20 }}
      animate={{ opacity: 1, y: 0 }}
      transition={{ delay, duration: 0.4 }}
      className="
        block bg-white rounded-2xl p-8
        shadow-sm hover:shadow-xl
        transition-all duration-300
        border border-gray-100
        hover:border-[#2A5A75]/20
        hover:-translate-y-1
        group
        text-center
      "
    >
      {/* الأيقونة */}
      <div 
        className="
          inline-flex items-center justify-center
          w-16 h-16 rounded-2xl
          mb-4
          transition-transform group-hover:scale-110
        "
        style={{ 
          backgroundColor: `${channel.color}15`,
        }}
      >
        <Icon 
          className="w-8 h-8" 
          style={{ color: channel.color }} 
        />
      </div>
      
      {/* العنوان */}
      <h3 className="text-xl font-bold text-[#1B3D4F] mb-2">
        {channel.title}
      </h3>
      
      {/* القيمة (الرقم/الإيميل) */}
      <p 
        className="text-lg font-semibold mb-3"
        style={{ color: channel.color, direction: 'ltr' }}
      >
        {channel.value}
      </p>
      
      {/* الوصف */}
      <p className="text-sm text-gray-600 leading-relaxed">
        {channel.description}
      </p>
    </motion.a>
  );
}
```

### تصميم SocialMediaIcons.tsx
```jsx
import { motion } from 'framer-motion';
import { 
  Instagram, 
  Twitter, 
  Facebook, 
  Youtube,
  Music2,    // TikTok placeholder
  Ghost      // Snapchat placeholder
} from 'lucide-react';

const iconMap = {
  Instagram,
  Twitter,
  Facebook,
  Youtube,
  Music2,
  Ghost,
};

export default function SocialMediaIcons({ channels }) {
  return (
    <div className="flex flex-wrap justify-center gap-4 max-w-3xl mx-auto">
      {channels.map((channel, index) => {
        const Icon = iconMap[channel.icon];
        
        return (
          <motion.a
            key={channel.id}
            href={channel.url}
            target="_blank"
            rel="noopener noreferrer"
            initial={{ opacity: 0, scale: 0.8 }}
            animate={{ opacity: 1, scale: 1 }}
            transition={{ delay: index * 0.05, duration: 0.3 }}
            whileHover={{ scale: 1.05, y: -4 }}
            className="
              group flex flex-col items-center
              w-28 h-28 p-4
              bg-white rounded-2xl
              shadow-sm hover:shadow-xl
              transition-shadow
              border border-gray-100
            "
            title={`${channel.name} - ${channel.handle}`}
          >
            <div 
              className="
                w-12 h-12 rounded-xl 
                flex items-center justify-center
                mb-2
                transition-transform group-hover:scale-110
              "
              style={{ 
                backgroundColor: `${channel.color}15`,
              }}
            >
              <Icon 
                className="w-6 h-6" 
                style={{ color: channel.color }}
              />
            </div>
            
            <span className="text-xs font-medium text-[#1B3D4F]">
              {channel.name}
            </span>
            
            <span className="text-[10px] text-gray-500 mt-0.5">
              {channel.handle}
            </span>
          </motion.a>
        );
      })}
    </div>
  );
}
```

## ملاحظات تقنية مهمة

### 1. أيقونات TikTok و Snapchat
lucide-react ما يحتوي أيقونات رسمية لـ TikTok و Snapchat. الحلول:

**الخيار أ (الأبسط):** استخدم placeholders من lucide:
- TikTok → `Music2`
- Snapchat → `Ghost`

**الخيار ب (الأفضل بصرياً):** استخدم SVG مخصصة من 
[simpleicons.org](https://simpleicons.org/) — مجانية ورسمية:
```jsx
// أيقونة TikTok
const TikTokIcon = ({ className, style }) => (
  <svg 
    className={className} 
    style={style}
    role="img" 
    viewBox="0 0 24 24" 
    fill="currentColor"
  >
    <path d="M12.525.02c1.31-.02 2.61-.01 3.91-.02..." />
  </svg>
);
```

**توصيتي:** ابدأ بالخيار أ (سريع)، لو مهم بصرياً اطلب من Replit يستخدم الخيار ب لاحقاً.

### 2. الأرقام بالـ LTR
الأرقام (920029007, 920013509) لازم تظهر بـ LTR حتى داخل صفحة RTL. 
استخدم `style={{ direction: 'ltr' }}` على العنصر.

### 3. روابط tel: و mailto:
- `tel:920029007` يفتح تطبيق الاتصال على الموبايل
- `mailto:csteam@baytelebaa.com` يفتح تطبيق البريد
- على الديسكتوب: قد يفتح تطبيق Skype/Mail الافتراضي

### 4. الواتساب
الرقم 920013509 لازم يكون مع كود الدولة الكامل في الرابط:
`https://wa.me/966920013509`

## التصميم العام

### الألوان:
- Primary: `#1B3D4F`
- Secondary: `#2A5A75`
- Background الصفحة: `#FAF7F2`
- Cards: white
- Border: `#E8DFD0`

### الخطوط:
- IBM Plex Sans Arabic أو Tajawal

### RTL:
- الصفحة كلها RTL
- الأرقام والإيميل والـ handles بـ LTR

### Responsive:
- Mobile: عمود واحد، أيقونات السوشيال 3 في الصف
- Tablet (md): قنوات التواصل 3 أعمدة
- Desktop (lg): نفس التابلت مع spacing أكبر
